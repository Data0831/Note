# All 拼接字串
***
Java
```Java
"as" + 12 + 8.7
```
>Java 可以透過 + 號拼接
- C++ 、C 、python 不行

C 、C++
```cpp
cout << "this is " << var << endl;
```

python
```python
f"this is {var1}"
```
>python 透過f-string與大括號{}來拼接字串
