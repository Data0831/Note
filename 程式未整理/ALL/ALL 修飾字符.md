# 修飾字符 modifier
***


- java
```java
class static_var{
	static int var = 10;
}
```
>static member variable 靜態成員變數
>讓同類型的物件共用同個變數
>自動初始化，不用等類別實體化就可以使用

#### 函式/方法
- C++ 、Java
```cpp
class static_func{
	static int func(){//初始化不用放外面
		return 10;
	}
}
```
