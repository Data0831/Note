# 創建與繼承
***
## 類別創建 + 物件繼承
C++:
```cpp
Class Base{//創建
};
Class Derived:Base{//繼承
};
```

Java:
```java
public class Base{
}
public class Derived extends Base{
}
```

Python:
```python
class Base:
    pass

class Derived(Base):
    pass
```
