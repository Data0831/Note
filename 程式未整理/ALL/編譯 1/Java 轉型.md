# Java 轉型
***

## 3 種轉型方法

1. 強制轉型（explicit casting）
```java
double d = (double) x;
```
2. 自動轉型（implicit casting）
```java
double d = 10;
```
3. 包装類別（Wrapper class）
```java
Integer i = 2;  // 自動裝箱
int y = i;  // 自動拆箱
```
