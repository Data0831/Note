## hashmap

- HashMap<K, V>
- hashmap.put(key, value)
- hashmap.get(key) -> value
- hashmap.containsKey(key) -> boolean

```java
private HashMap<Character, Character> map = new HashMap<>();
map.put('(', ')');
map.get('(');
```
