# Python 大小寫
***
>[!info]+
>string.captalize(): 句子首字母大寫
>string.title(): 單詞首字母大寫
>string.swapcase(): 大小寫交換
>string.lower()
>string.casefold() -> 更強大的lower()
>string.upper()

```python
txt = "hello world!!!"
txt.capitalize() # "Hello world!!!"
txt.title() # "Hello World!!!"
txt.swapcase() # "HELLO WORLD!!!"
```
