# visual studio 使用方式
透過反組譯觀察組譯數量，與內存存取數量，來判斷程式的效率

>[!note]+
>| 功用         | 按鍵               |
|:-----------|:-----------------|
| 執行         | Ctrl + F5        |
| debug      | F5               |
| STOP debug | shift + F5       |
| 反組譯        | ALT+G            |
| format     | Ctrl + k, d      |
| 摺疊/展開      | Ctrl + M, M      |
| 摺疊所有       | Ctrl + M,&nbsp;O |
| 展開所有       | Ctrl + M,&nbsp;L |    


debug: 很多調適信息
release:去除調適信息，生成執行文件經過優化
- 去除優化->對著專案右鍵->屬性->左方優化

## EOF 檔案末尾
在visual studio "^z^z^z" -> ctrl+z 三次為檔案末尾
